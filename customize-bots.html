How to customize bots messages

Bots are pretty fun, but sometimes you may think "it would be nice to be able to edit bots messages!", well, you can do it right now.




Requirements

  • JPEXS

  • bots SWF

  • Java oracle


Step 1. Open bots in JPEXS

Click "Open" button and locate bots.




Step 2. Moddify.

After opening your SWF start editing it. Open "Main" file in this SWF and scroll to network.chat . Click "Edit ActionScript button at the bottom, edit your message and save.




Step 3. Saving the SWF

Click "save as" and choose any name for it. Use save as instead of save to dont lose original file and be able to edit it again.
